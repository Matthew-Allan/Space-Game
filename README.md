## A game set in space.
Created with SDL2 and OpenGL in C.

Now supported by Mac and Windows!

### For Mac 
If compiling, you will need to have SDL2 installed using Homebrew.

This can be done as so:
```
brew install SDL2
```

### For Windows
This should compile out the box however you will need GNU make as well as GCC to use the provided makefile. All binaries and assets that are requires should be provided.

### Building
Running ```make``` should build and run it as long as the prerequisites are met.
